How to create the bitmap images

Create PNG from SVG

  1. Open the file bndtools-swirl.svg with Inkspace (I'm using version 0.48.1)
  2. Select in the menu: File->Export Bitmap
  3. Set the bitmap width to 2048 and the filename to swirl-2048.png
  4. Press the Export button
You now have the large version of the bitmap.

Create PNG image copy with glow

  1. Open the file swirl-2048.png with GIMP (I'm using version 2.6.11)
  2. Select in the menu: Filters->Light and Shadow->Gradient Flare...
  3. Select the Default entry on the Selector tab, as shown below
  4. Use the settings as shown below
  5. Select in the menu: File->Save As...
  6. Set the filename to swirl-glow-2048.png and press the Save button; use the save settings as shown below
You now have the large versions (regular and glow) of the bitmaps.

Scale images

  1. Open the file swirl-2048.png with GIMP
  2. Select in the menu: Image->Scale Image...
  3. Set the width to 512 and press the Scale button
  4. Select in the menu: File->Save As...
  5. Set the filename to swirl-512.png and press the Save button; use the save settings as shown below
  6. Select in the menu: Edit->Undo Scale Image
  7. Repeat (go to step 2 of the procedure) for all other sizes
  8. Repeat the complete procedure for the swirl-glow-2048.png file

Create 512x512 images

  1. Open the file swirl-512.png with GIMP
  2. Select in the menu: Image->Canvas Size...
  3. Click he chain icon next to the Width and Height fields to unlock the fields, set the Height to 512, click the Center button, and finally click the Resize button
  4. Select in the menu: File->Save As...
  5. Set the filename to swirl-512x512.png and press the Save button; use the save settings as shown below
  6. Repeat the above procedure for the swirl-glow-512.png file

Create Eclipse welcome page icons

  1. Open the file swirl-48.png with GIMP
  2. Select in the menu: File->Save As...
  3. Set the filename to bndtools.gif and press the Save button, press the Export button, and finally press the Save button
  4. Repeat the above procedure for the file, but name it bndtools-glow.gif
  5. Move both these gif images into the bndtools.core/resources/intro/images directory
Done!
